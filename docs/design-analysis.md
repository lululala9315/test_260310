# Apple Mac Mini 사이트 디자인 분석서

> 분석 대상: https://www.apple.com/kr/mac-mini/
> 분석 목적: NEO Home 로봇 가사 도우미 구독 플랫폼 랜딩페이지 제작 레퍼런스

---

## 1. 색상 팔레트

| 요소 | 색상값 | 용도 |
|------|--------|------|
| 배경 (밝은) | #FFFFFF | 메인 배경 |
| 배경 (어두운) | #000000 / #1D1D1D | Hero, 강조 섹션 |
| 텍스트 (주) | #1D1D1F | 헤드라인, 본문 |
| 텍스트 (보조) | #86868B | 서브 텍스트, 설명 |
| 강조색 (CTA) | #0071E3 | 버튼, 링크 |
| 강조색 (hover) | #0077ED | 호버 상태 |
| 배경 (카드) | #F5F5F7 | 밝은 회색 카드 배경 |

**특징**: 최소한의 중성 색상 + 단일 강조색(파란색)으로 제품에 집중시키는 전략

## 2. 타이포그래피

### 폰트 패밀리
`-apple-system, BlinkMacSystemFont, "Segoe UI", Helvetica, Arial, sans-serif`

### 크기 계층 구조

| 요소 | 크기 | 무게 | 라인높이 |
|------|------|------|---------|
| Hero 헤드라인 | 48-64px | Bold (700) | 1.1-1.2 |
| 섹션 타이틀 | 28-36px | Semibold (600) | 1.2-1.3 |
| 서브 헤드라인 | 18-24px | Regular (400) | 1.4 |
| 본문 | 14-16px | Regular (400) | 1.5-1.6 |
| 소형 텍스트 | 12-13px | Regular (400) | 1.4 |

**특징**: `clamp()` 함수로 반응형 폰트 스케일링, 헤드라인과 본문 간 명확한 대비

## 3. 레이아웃 패턴

### 전체 구조
```
[고정 네비게이션 헤더]
[Hero - 풀 너비, 다크 배경, 중앙 정렬 텍스트 + 제품 이미지]
[기능 소개 - 밝은 배경, 그리드 카드 레이아웃]
[스펙/비교 - 테이블 또는 그리드]
[CTA - 중앙 정렬]
[푸터 - 간결한 링크]
```

### 간격 규칙
- **섹션 간 여백**: 60-100px (clamp 적용)
- **콘텐츠 최대폭**: 980px (중앙 정렬)
- **좌우 패딩**: 모바일 20px / 데스크톱 40-60px
- **컴포넌트 내부**: 16-32px

---

## 4. UI 컴포넌트

### 버튼
- **Primary CTA**: 파란 배경(#0071E3), 흰 텍스트, 완전 둥근 모서리(pill), 14-16px
- **Secondary**: 파란 텍스트 링크, 밑줄 또는 화살표 아이콘
- **호버**: 200ms ease transition, 미묘한 색상 변화

### 카드
- 둥근 모서리(12-16px border-radius)
- 그림자 없음 (Apple 특유의 플랫 디자인)
- 호버 시 미세한 상승 효과(translateY: -4px)

### 네비게이션
- 고정(fixed), 반투명 배경 + backdrop-blur
- 높이 44-48px, 간결한 텍스트 링크

---

## 5. 애니메이션 & 인터랙션

### 스크롤 트리거 애니메이션
- **패턴**: opacity 0→1 + translateY 20-30px→0
- **시간**: 600-800ms ease
- **방식**: IntersectionObserver API (성능 최적화)
- **실행**: 한 번만 (진입 시)

### 로드 애니메이션 (Hero)
- 요소별 순차 등장 (stagger delay 0.1-0.2s 간격)
- 헤드라인 → 서브 텍스트 → 버튼 → 이미지 순서

### 호버 상태
- 버튼: 색상 변화 (200ms)
- 카드: translateY(-4px) (200ms)
- 링크: 밑줄 등장

## 6. 이미지 & 비주얼

- **Hero**: 초고해상도 제품 이미지, 중앙 배치, 다크 배경 위
- **기능 카드**: 아이콘 또는 작은 일러스트
- **포맷**: WebP 우선, JPEG 폴백
- **최적화**: lazy loading, responsive srcset, picture 태그

## 7. 디자인 무드 & 톤

### 키워드
**"우아한 미니멀리즘 + 기술 혁신"**

### 특성
- 넓은 여백으로 고급스러움 연출
- 제품/서비스가 주인공 — UI 요소는 최소화
- 기술 스펙을 감성적으로 전달 (숫자보다 가치 강조)
- 짧고 임팩트 있는 카피 (한 문장에 하나의 메시지)

## 8. NEO Home 랜딩페이지 적용 요약

| Apple 디자인 요소 | NEO Home 적용 방식 |
|------------------|-------------------|
| 다크 Hero + 제품 중앙 | 다크 배경에 NEO 로봇 비주얼 중앙 배치 |
| pill 버튼 CTA | 파란 둥근 "구독 시작하기" 버튼 |
| 밝은/어두운 섹션 교차 | Hero(다크) → 기능(밝은) → 가격(다크) → CTA(밝은) |
| 미니멀 카드 레이아웃 | 기능 4개 그리드, 가격 3개 그리드 |
| fadeInUp 스크롤 애니메이션 | 모든 섹션에 IntersectionObserver 기반 적용 |
| 반투명 고정 네비게이션 | backdrop-blur 네비게이션 바 |

## 디자인 추천

제공된 링크(Apple Mac Mini) 디자인이 NEO Home 랜딩페이지에 **가장 적합**합니다.

**이유:**
1. **제품 중심 프레젠테이션**: 단일 제품(NEO 로봇)을 강조하는 구조와 일치
2. **프리미엄 포지셔닝**: 고가 구독 서비스의 신뢰감을 줄 수 있는 미니멀 고급 디자인
3. **다크/라이트 교차 레이아웃**: 구독 플랜 비교에 효과적인 시각적 구분
4. **기술 + 감성 밸런스**: 로봇 스펙을 딱딱하지 않게 전달하는 Apple 특유의 카피 스타일
